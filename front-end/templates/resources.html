    • Styling Resources

      • Docs for Bootstrap
      • Bootstrap is written in LESS, a terrific "dynamic stylesheet language" that lets you use variables, mixins, operations, and functions with CSS. If you want to do a lot of styling work, I'd recommend considering LESS in conjunction with WinLess, which compiles all your LESS to CSS files every time you hit save. Makes styling much easier, in my opinion.
      • Foundation, an alternative to bootstrap
      • Compass, which is based on Sass (an alternative to Less), is another good framework (or so I'm told) for authoring solid CSS, with a lot of great stuff built in.
      • If responsive design is really what you're all about, (ie; you want to do a great job on lots of different screen sizes), definitely check out 320+Susy for a remarkable example (resize your window on it! lots!) and some good resources.
